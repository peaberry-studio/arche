# Workspace Config Source

This directory stores legacy/source material for workspace runtime configuration.

- `CommonWorkspaceConfig.json`: reference config used as source material.

Deployment:

- `scripts/deploy-config.sh` now initializes `kb-config` as an empty bare repository.
- Runtime config is generated by kickstart apply (`/u/[slug]/kickstart`) and written to
  `kb-config/CommonWorkspaceConfig.json`.
